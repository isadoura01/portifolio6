PROJETO 2 TRIMESTE: ASSIM QUE ACABA

Esse projeto de site foi montado em algumas etapas básicas, começando pelo planejamento do que seria o conteúdo e o visual. Usamos HTML para estruturar as páginas e CSS para dar estilo, deixando o visual mais limpo e fácil de navegar. Uma coisa legal é que o projeto foi inspirado no site "AluraBooks," que criamos durante as aulas na Alura, onde tivemos orientações passo a passo para organizar o código e melhorar o design.

O site foi inspirado na duologia emocional criada pela autora Colleen Hoover. Tudo começa com o livro *É Assim que Acaba*, que conta a história de Lily Bloom, uma jovem que enfrenta relacionamentos complexos e abusivos enquanto lida com traumas do passado. O livro se tornou muito popular por sua abordagem sensível de temas difíceis, como violência doméstica e superação pessoal.

O filme, que leva o mesmo nome, adapta a história de Lily para as telas, explorando os desafios emocionais e morais que ela enfrenta.

O segundo livro, *É Assim que Começa*, é uma continuação direta da história e se aprofunda no relacionamento de Lily com Atlas Corrigan, seu amor de infância. Ele expande a jornada de superação, mostrando como as experiências do passado influenciam suas escolhas para o futuro. Essas obras trazem temas fortes e emotivos, que ressoam com leitores e espectadores pela honestidade com que tratam o amor, a dor e o processo de recomeçar.

PROJETO 3 TRIMESTRE: MANUAL DE ASSASSINATO PARA BOAS

O Manual de Assassinato para Boas Garotas é um site desenvolvido com o objetivo de ajudar as pessoas a estudar e aprender sobre o livro e a série de mesmo nome. O projeto foi criado de acordo com as instruções da plataforma Alura, utilizando o site Flashcards como base. A ideia é fornecer uma maneira divertida e interessante de explorar os temas da obra, ao mesmo tempo em que permite aos usuários revisar e adquirir novos conhecimentos de forma prática.

O design do site é responsivo, ou seja, ele se adapta bem a diferentes dispositivos, como computadores, tablets e celulares, proporcionando uma navegação fluida e agradável. A escolha das cores foi feita para tornar o site visualmente atraente, sem sobrecarregar o usuário. A paleta de cores foi pensada para ser funcional e agradável aos olhos, facilitando a leitura e a exploração dos conteúdos.

O tema do site é inspirado no livro e na série "Manual de Assassinato para Boas Garotas", com elementos que remetem à história de forma sutil e criativa. Isso torna o site não apenas uma ferramenta de estudo, mas também uma forma divertida de aprender mais sobre a obra, conectando os usuários à narrativa de maneira interativa. O site foi desenvolvido com HTML, CSS e JavaScript. O HTML organiza o conteúdo, enquanto o CSS define o estilo visual e o layout, e o JavaScript adiciona interatividade, permitindo que os usuários naveguem facilmente pelos flashcards e explorem os temas relacionados ao livro. O resultado é um site funcional, com um design atrativo e que cumpre seu objetivo de ajudar no aprendizado de uma forma envolvente e acessível.
